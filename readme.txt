Make Utility
>>
Program Build: make main
Program Build with Debug: make main_debug
Build Reset: make clean

Run the program
>>
./main

Main Menu
>>
1. Create: 새로운 가게 정보 기록 기능
2. Read: 기록되어있는 모든 가게들의 정보 출력 기능
3. Update: 기존에 기록되어있는 가게의 정보 수정 기능
4. Delete: 기존에 기록된 가게 중 원하는 가게의 기록 삭제 기능
5. Search(name): 원하는 가게의 정보를 찾아 출력 기능
6. Search(time): 원하는 배달시간의 가게를 모두 찾아 그 정보를 출력 기능
7. Search(ctgr): 원하는 카테고리의 가게를 모두 찾아 그 정보를 출력 기능
8. Delete(condition): 원하는 조건의 가게 모두 삭제 기능(category 또는 time에 따라)
9. Delete(all): 현재 기록되어있는 모든 가게 정보 삭제 기능
10. Load: txt file 내 모든 가게의 정보를 읽어와서 프로그램에 기록하는 기능
11. Save: program 내 모든 가게 정보를 txt file에 저장 기능
0. Quit: program 종료 기능
